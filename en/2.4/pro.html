Security Onion Pro

In 2022, we announced that we would be releasing enterprise features that would only be available to paid users of the platform. You can read the announcement at https://blog.securityonion.net/2022/08/security-onion-enterprise-features-and.html.

Starting in Security Onion 2.4.70, licensed users of Security Onion Pro can activate the following features: OpenID Connect (OIDC) 3rd-party authentication, LUKS disk encryption, FIPS OS compliance, STIG OS compliance, Notifications, and time tracking inside of Cases.

Starting in Security Onion 2.4.80, licensed Pro users can also enable Guaranteed Message Delivery.

Starting in Security Onion 2.4.120, licensed Pro users can also enable Security Onion Connect API.

Starting in Security Onion 2.4.130, licensed Pro users can also access Active Query Management.

Starting in Security Onion 2.4.150, licensed Pro users can also access Manager of Managers (MoM).

Starting in Security Onion 2.4.160, licensed Pro users can also access MCP Server and Security Onion App for Splunk.

Starting in Security Onion 2.4.170, licensed Pro users can also create a Hypervisor node to run Security Onion virtual machines.

Starting in Security Onion 2.4.180, licensed Pro users can also generate Reports.

Starting in Security Onion 2.4.190, licensed Pro users can also access the Onion AI Assistant.

Note

Contact Security Onion Solutions, LLC via our website at https://securityonion.com/pro for more information about purchasing a Security Onion Pro license to enable these features.
